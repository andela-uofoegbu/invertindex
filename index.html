B-Index 1.0

B-Index 1.0 will take JSON file(s) containing book title and text, and produce a list of words indicating which books contain the words

How To Use

Click 'Upload' button to select JSON file(s)
Click Create Index
To search for a word, enter a word and click 'Search'
To clear current files click 'Reset Files'
